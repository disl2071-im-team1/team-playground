Three readings of the same air, and they don't agree, one isn't even a concentration. There is no single true air-quality number: who measured it, with what instrument, and on what scale all change the answer. An AQI index is not µg/m³, so you can't read these as one figure or compare them directly. The source matters as much as the number.

How the data moves
Three independent sensor networks, SMHI (official), WAQI (commercial), and luftdaten (citizen), are polled over the internet once an hour by the integration layer. It normalises every reading to one schema, appends it to a shared append-only store (the data branch), and serves it through a read API that this app reads. The integration layer is a central server, and it is in the path: it is the core of the flow. There is no mesh, no phone, and no on-device routing in this prototype.
